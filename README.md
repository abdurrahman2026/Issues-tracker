1 / var is function scoped and can be redeclared, let is block scoped and can change value, 
    and const is block scoped but its value cannot be changed after assignment.

2/ The spread operator (...) is used to expand elements of an array or object into individual elements.

3/ map(), filter(), and forEach() are array methods used to loop through elements.
    map() and filter() return a new array, but forEach() only runs a loop and does not return anything.

4/ An arrow function is a shorter way to write a function in JavaScript using =>. 
    It makes the code more simple and clean compared to normal functions.

5/ Template literals are a way to write strings in JavaScript using backticks ( ).
    They allow us to easily add variables inside a string using ${}.
